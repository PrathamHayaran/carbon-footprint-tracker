# 🌍 Carbon Footprint Tracker

A modern web application that helps users calculate, monitor, and reduce their carbon footprint by tracking daily activities and providing sustainability insights.

## 🚀 Features

- 📊 Track carbon emissions from daily activities
- 🎯 Set sustainability goals
- 📈 Visualize emission trends with interactive charts
- 💡 Receive personalized insights and recommendations
- 📱 Responsive design for desktop and mobile devices
- ⚡ Fast and modern user experience built with Next.js

## 🛠️ Tech Stack

- **Frontend:** Next.js, React, TypeScript
- **Styling:** CSS
- **State Management:** React Context API
- **Charts & Visualization:** Custom Trend Charts
- **Deployment:** Docker, Google Cloud Build

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/PrathamHayaran/carbon-footprint-tracker.git
```

### Navigate to the project

```bash
cd carbon-footprint-tracker
```

### Install dependencies

```bash
npm install
```

### Run the development server

```bash
npm run dev
```

Open your browser and visit:

```bash
http://localhost:3000
```

## 🎯 How It Works

1. Users input their daily activities.
2. The application calculates estimated carbon emissions.
3. Data is displayed through visual dashboards and charts.
4. Users can set emission reduction goals.
5. The system provides insights to encourage sustainable habits.

## 🌱 Impact

The Carbon Footprint Tracker aims to promote environmental awareness by helping individuals understand how their daily choices affect the planet and encouraging sustainable living practices.
